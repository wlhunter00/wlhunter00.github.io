Practicum

  • May 2020

  • Co-Founder

  • Personal Startup

Problem

Students can only work in an agile environment through an internship, but COVID-19 canceled many of these opportunities. Without the formal structure of a company, it is difficult for CS students to work on a meaningful project.

Product

Practicum is a program designed to place students on teams that operate as a startup. The projects are sourced from entrepreneurs that need help developing an MVP. Teams have mentors that can guide the students through development.

Process

  • Developed a recruitment pipeline, which generated 300+ applications and 15+ clients in two weeks. Interviewed students to find pain points.
  • Lead a team of 8 partners to build the curriculum. The primary KPI was conversion rate between interview rounds, which averaged 88%.
  • Vetted potential clients to ensure quality, maintaining a 33% acceptance rate. Flushed out features of products for student development teams.

Project Covid

  • March 2020

  • Co-Founder

  • LFR International

Problem

Information regarding COVID-19 is incredibly scattered, and more detailed sources are not user-friendly. There is no centralized, trustworthy location where a person can access information, statistics, and other resources quickly.

Product

Project Covid is a mobile app that aggregates vetted information regarding the pandemic. The app includes statistics, the latest news, educational toolkits, and local testing information and presents all sources used by the team.

Process

  • Wrote 50+ user stories and prioritized which features the team should pursue, met with stakeholders and users to understand the use cases.
  • Managed a team of 12 engineers, assigned user stories and resolved roadblocks. Wrote majority of Express server used by app.
  • Oversaw launch of the app on TestFlight. Received feedback from 80+ beta testers and overhauled the home screen.

Recruitment Reddit Bot

  • January 2020

  • Risk Manager

  • Alpha Delta Phi

Problem

The early stages of my organization’s recruitment cycle are very reliant on qualitative observations about potential candidates. With a large number of students to consider, there was no quantitative measure of fit for the organization.

Product

The Recruitment Reddit Bot is a python bot that gathers data regarding comments members have left about candidates in a private subreddit. It assigns sentiment to the messages and then calculates the traction a candidate has within the organization.

Process

  • Utilized Reddit Praw Python API to parse Reddit 1200+ comments using Regex. Parsed keywords to determine sentiment of feedback.
  • The bot used multiple metrics such as upvotes, comment sentiment, and engagement to calculate an excitement score for a candidate.
  • Automatically populated a Google Sheets document so the recruitment team could track the statistics generated.
